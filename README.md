# Directory Inspector

A simple Python CLI tool that analyzes the contents of a folder.

## Features

- Lists files grouped by extension
- Displays file sizes
- Shows total number of files and folders
- Detects the largest file
- Handles files without extension

## Usage

Run the script:

```bash
python inspector.py
```

Then enter the folder path when prompted.
